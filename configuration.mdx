### GitHub Integration

Automatically sync your documentation with GitHub repositories.

<CodeGroup tabs="Webhook Setup,Manual Sync">
  ```javascript
// Webhook configuration
const webhookConfig = {
  url: 'https://nullish.studio/webhooks/github',
  events: ['push', 'pull_request'],
  secret: 'your_webhook_secret'
};

fetch('https://api.github.com/repos/owner/repo/hooks', {
  method: 'POST',
  headers: {
    'Authorization': `token ${GITHUB_TOKEN}`,
    'Content-Type': 'application/json'
  },
  body: JSON.stringify(webhookConfig)
});
```
  ```bash
# Manual sync script
#!/bin/bash

# Pull latest changes
git pull origin main

# Build and deploy documentation
npm run build
npm run deploy

echo "Documentation updated successfully"
```
</CodeGroup>

## Configuration Validation

Before applying changes, validate your configuration to catch potential issues.

```javascript
const validateConfig = (config) => {
  const errors = [];
  
  if (!config.name) errors.push('Project name is required');
  if (!config.primaryColor.match(/^#[0-9A-F]{6}$/i)) {
    errors.push('Invalid primary color format');
  }
  if (config.rateLimit < 100) errors.push('Rate limit too low');
  
  return errors;
};

// Example usage
const myConfig = {
  name: 'My Project',
  primaryColor: '#3B82F6',
  rateLimit: 1000
};

const validationErrors = validateConfig(myConfig);
if (validationErrors.length > 0) {
  console.error('Configuration errors:', validationErrors);
}
```
