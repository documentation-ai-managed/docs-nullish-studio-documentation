## Release History

<Update label="2024-10-15" description="v2.1.0" tags={["feature", "improvement"]}>
## New Features

- Added support for custom domains with automatic SSL certificate provisioning
- Introduced advanced search with filters for content type and date range
- Implemented collaborative editing with real-time cursor tracking

## Improvements

- Enhanced API performance with response caching and optimized database queries
- Improved mobile responsiveness across all documentation pages
- Updated editor with better syntax highlighting for multiple programming languages
</Update>

<Update label="2024-09-20" description="v2.0.0" tags={["feature", "breaking"]}>
## New Features

- Launched redesigned user interface with modern design system
- Added webhook support for real-time event notifications
- Introduced team management with role-based access control

## Breaking Changes

- API endpoints now require authentication headers for all requests
- Deprecated legacy authentication methods in favor of OAuth 2.0
- Updated minimum supported browser versions for enhanced security
</Update>

<Update label="2024-08-10" description="v1.5.0" tags={["feature", "bugfix"]}>
## New Features

- Added GitHub integration for automatic documentation syncing
- Introduced API key management with granular permissions
- Launched community forum for user discussions and support

## Bug Fixes

- Fixed rendering issues with complex code blocks in documentation
- Resolved authentication timeout problems for long-running sessions
- Corrected search indexing to include recently updated content
</Update>
